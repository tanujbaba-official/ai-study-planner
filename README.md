# 📅 AI Study Planner

**Smart Study Planner with Calendar, Admin Panel, PYQ/Syllabus/Book PDFs, Progress Tracker & Pomodoro Timer**

Built for students who want everything in one place — syllabus, exam dates, PYQs, books, smart timetable aur daily progress.

---

### ✨ Features

- **Admin Panel** (Password Protected)
  - Add / Edit / Delete Subjects
  - Upload Syllabus, PYQs & Book PDFs
- **Exam Calendar** with color-coded urgency
- **AI Smart Timetable** (auto generates 7-day plan)
- **Progress Tracker** (topic-wise checkbox)
- **Pomodoro Timer** (25 min study + break)
- **Daily AI Study Tips**
- **PDF Download** directly from app (Syllabus + PYQ + Books)
- 100% Local data (no login, no cloud)

---

### 🚀 How to Use

1. **Admin Login** → Enter Password: `admin123` (Can be changed Anytime)
2. Subjects add karo + Upload PDFs
3. Progress Tracker mein daily tick mark karo
4. Click On AI Timetable button → automatic plan will generate
